# 🎋 五行小宝 · 人生游戏说明书（WXTI）

> 测测你的灵魂出厂配置 · 60 日柱人格底色全览

基于八字命理「日柱」切入，融合 MBTI 式自嘲鬼畜文案的趣味人格测试。

## ✨ 特性

- 🎯 **60 种人格底色**：基于 60 甲子日柱，每种都有独特"人格名 + 灵魂弹幕 + 出厂技能 + 日柱解读"
- 🎨 **五行主色动态切换**：木绿 / 火红 / 土黄 / 金银 / 水蓝，每个人看到的结果页都是自己的颜色
- 🤝 **天选搭子 / ⚠️ 警惕对象**：基于地支三合六合六冲六害，告诉你跟谁合得来
- 💘 **测测 TA**：输入 TA 的生日生成"你 × TA"双人配对卡
- 📸 **1080×1920 高清分享卡**：一键导出，适配小红书/朋友圈
- 🎨 **MBTI 潮酷风视觉**：暗黑禅意 × 手写字体 × 荧光色反差

## 🏗️ 技术栈

纯静态 HTML + CSS + JavaScript，零依赖（除了 html2canvas 从 CDN 加载用于分享卡导出）。

```
wxti-h5/
├── index.html              # 主页面
├── css/style.css           # 样式
├── js/
│   ├── ganzhi.js           # 天干地支 / 四柱八字算法
│   ├── data.js             # 60 日柱人格数据
│   ├── pair.js             # 配对关系数据
│   ├── img-manifest.js     # 人格图索引
│   └── app.js              # 主逻辑
└── assets/
    ├── logo.png            # 品牌 logo
    ├── personas-y2k/       # 60 张 Y2K 风人格图
    └── wechat-qr.jpg       # 公众号二维码
```
